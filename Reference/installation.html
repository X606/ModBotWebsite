How to install mod-bot!

First of all, make sure you're not in a beta build

Then, follow the following steps:

  1. Download the Mod-Bot laucher from here
  2. Open the downloaded zip file with the windows explorer
  3. Drag the ModBot Launcher.exe file onto your desktop
  4. Run the ModBot Launcher.exe file, it will prompt you for admin privilages, click yes
  5. You should now see something like this
  6. Click on "Install Mod-Bot" in the lower right corner and wait for it to finish loading
  7. The Launcher should now look something like this, simply press "start game" and it will automatically start the game for you!

In the future, try to start clone drone though the laucher when you want to play with mod-bot, this will make sure everything is up to date!
